# Homem-Aranha - Página Web

## Descrição
Este é um projeto de página web dedicada ao Homem-Aranha, focando em Miles Morales. A página apresenta informações sobre o personagem, sinopse, elenco e detalhes do filme, com design responsivo para diferentes dispositivos.

## Tecnologias Utilizadas
- **HTML5**: Estrutura da página
- **CSS3**: Estilização e responsividade
- **Google Fonts**: Fonte Plus Jakarta Sans

## Como Executar
1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/homem-aranha.git
   ```
2. Abra o arquivo `index.html` em um navegador web.

## Estrutura do Projeto
```
homem-aranha/
├── index.html          # Arquivo principal HTML
├── src/
│   ├── css/
│   │   ├── reset.css   # Reset de estilos
│   │   ├── style.css   # Estilos principais
│   │   └── responsivo.css # Estilos responsivos
│   ├── fonts/          # Fontes (se houver)
│   └── image/          # Imagens do projeto
└── README.md           # Este arquivo
```

## Responsividade
A página é responsiva e se adapta a telas de:
- Desktop (acima de 1024px)
- Tablets (até 1024px)
- Celulares (até 768px)
