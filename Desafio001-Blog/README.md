# Blog CodeLab

Um blog simples sobre tecnologia, programação e novidades do mundo digital.

## Descrição

Este projeto é um blog estático desenvolvido em HTML e CSS, focado em conteúdos relacionados à tecnologia, programação e inovações digitais. O layout é responsivo e utiliza fontes modernas e ícones do Font Awesome para uma experiência visual agradável.

## Estrutura do Projeto

```
index.html                 # Página principal do blog
src/
  css/
    reset.css             # Reset de estilos CSS
    responsivo.css        # Estilos responsivos para diferentes dispositivos
    style.css             # Estilos principais
  img/                    # Diretório para imagens
README.md                 # Este arquivo
```

## Como Usar

1. Clone ou baixe o repositório.
2. Abra o arquivo `index.html` em qualquer navegador web moderno.
3. O blog estará pronto para visualização.

## Tecnologias Utilizadas

- **HTML5**: Estrutura da página
- **CSS3**: Estilização e responsividade
- **Font Awesome**: Ícones
- **Google Fonts**: Tipografia (Space Grotesk e Inter)

## Responsividade

O blog é totalmente responsivo, adaptando-se a diferentes tamanhos de tela:
- Desktop
- Tablet
- Mobile

Utiliza media queries no arquivo [`src/css/responsivo.css`](src/css/responsivo.css) para garantir uma boa experiência em todos os dispositivos.

## Funcionalidades

- Header com título e campo de pesquisa
- Seção de cards para posts do blog
- Layout limpo e moderno
