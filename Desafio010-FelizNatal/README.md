# Projeto de front-end desenvolvido como exercício do Desafios Codelab.

## Sobre

Uma landing page temática de Natal criada em HTML e CSS, com layout responsivo e foco em uma plataforma de doação para espalhar alegria durante as festas.

## Tecnologias

- HTML5
- CSS3

## Estrutura do projeto

- `index.html` - página principal
- `src/css/reset.css` - reset de estilos padrão do navegador
- `src/css/style.css` - estilos principais
- `src/css/responsivo.css` - estilos responsivos para dispositivos menores
- `src/img/` - imagens e ícones utilizados no layout

## Observações

- O layout é adaptado para telas de desktop e mobile.
- Os links de redes sociais e botões usam URLs de exemplo.
