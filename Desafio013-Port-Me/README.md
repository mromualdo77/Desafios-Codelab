# Portfólio - Maria Romualdo

Este projeto é um portfólio pessoal simples e responsivo criado em HTML e CSS. Ele apresenta seções de apresentação, sobre mim, experiência e projetos, com navegação interna e estilo escuro.

## Estrutura do Projeto

- `index.html` - página principal do portfólio.
- `src/css/reset.css` - redefinição padrão de estilos para garantir compatibilidade entre navegadores.
- `src/css/style.css` - estilos principais do layout, tipografia, botões e componentes.
- `src/css/responsivo.css` - regras de responsividade para exibição em tablets e celulares.
- `src/image/` - pasta para imagens usadas no layout (como foto de perfil e ilustrações de projetos).

## Tecnologias

- HTML5
- CSS3

## Seções do Portfólio

- Início
- Sobre mim
- Experiência
- Projetos
- Rodapé

## Uso

1. Abra o arquivo `index.html` em um navegador.
2. Navegue pelo menu para acessar as seções da página.

## Personalização

- Atualize os textos com suas informações pessoais.
- Substitua a imagem de perfil em `src/image/me.png` por sua própria foto.
- Ajuste os cards de projetos com nomes, descrições e links reais.

## Observações

- O layout já possui adaptação para telas menores usando as regras em `src/css/responsivo.css`.
- Este repositório é ideal como base para um portfólio frontend estático.
