# 🐕 HelpDog

## Descrição

HelpDog é uma plataforma web dedicada a ajudar cães em situação de rua. O projeto tem como objetivo conectar pessoas dispostas a fazer doações com iniciativas de cuidado e proteção animal, promovendo a conscientização sobre bem-estar animal.

## 🎯 Objetivo

- Facilitar doações para ajudar cães necessitados
- Divulgar histórias inspiradoras sobre cães resgatados
- Conectar pessoas que querem contribuir para a causa animal
- Promover adoções responsáveis

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura semântica da página
- **CSS3** - Estilização e responsividade
- **Mobile First** - Design responsivo para todos os dispositivos

## 🎨 Seções da Página

### 1. **Banner Principal**
- Logo da comunidade
- Título: "Ajude quem realmente precisa da sua ajuda"
- Call-to-action para doações
- Imagem de destaque com cão

### 2. **Seção Destaque**
- Conteúdo inspirador sobre a conexão entre humanos e cães
- Imagem complementar
- Botão "Saiba mais"

### 3. **Seção de Doações**
Apresenta 3 opções de produtos para doação:
- **Special Dog** - Ração completa com proteínas e vitaminas
- **Boscrok** - Biscoitos com cálcio e ômega 6
- **Drools** - Alimento com fígado de frango

### 4. **Rodapé**
- Créditos do desafio (CodeLab)
- Desenvolvedor
- Copyright

## 📱 Design Responsivo

O site utiliza três arquivos CSS para garantir uma experiência ótima em todos os dispositivos:
- `reset.css` - Remove inconsistências entre navegadores
- `responsivo.css` - Adaptações para diferentes tamanhos de tela
- `style.css` - Estilização principal e responsividade

## 🚀 Como Executar

1. Clone ou baixe os arquivos do projeto
2. Abra o arquivo `index.html` em seu navegador
3. Pronto! A página está funcionando

Não há dependências externas ou etapas de instalação complexas.
